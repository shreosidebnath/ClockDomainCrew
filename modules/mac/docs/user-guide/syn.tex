\subsection{SDC File}
An \texttt{.sdc} file is generated to provide synthesis and static timing
analysis tools guidance for synthesis.

The \texttt{Mac.sdc} file can be found in the test case directory as follows: \\
\texttt{./modules/mac/generated/synTestCases/<test case name>/Mac.sdc}

\subsection{Multicycle Paths}
None. The design is architected to complete all framing, CRC generation, and lane-alignment logic within a single clock cycle at 156.25 MHz.
