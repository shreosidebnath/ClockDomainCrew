\section{Port Descriptions}

The ports for the \textbf{Pcs} core are shown below in Table~\ref{table:ports}. The width of several ports is controlled by the following parameters:

\begin{itemize}[noitemsep]
  \item \textit{dataW} is the width of the internal data bus (typically 64 for 10GBASE-R)
  \item \textit{ctrlW} is the width of the XGMII control bus, equal to \textit{dataW}/8
  \item \textit{hdrW} is the width of the 64b/66b sync header (fixed at 2 bits)
\end{itemize}
