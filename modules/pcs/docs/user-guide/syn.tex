\subsection{SDC File}
An \texttt{.sdc} file is generated to provide synthesis and static timing
analysis tools guidance for synthesis.

The \texttt{Pcs.sdc} file can be found in the test case directory as follows: \\
\texttt{./modules/pcs/generated/synTestCases/<test case name>/Pcs.sdc}

\subsection{Multicycle Paths}
None. The design is architected to complete all encoding, scrambling, and residue checks within a single clock cycle at 156.25 MHz.
